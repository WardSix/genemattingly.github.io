# Video Loops

Add the muted loop snippets referenced by `index.html` here:

- `hero-loop.mp4`
- `about-loop.mp4`
- `portfolio-loop.mp4`
- `pricing-loop.mp4`
- `contact-loop.mp4`

Use short, vertically-friendly clips to keep page weight low. Each video should be encoded for web playback (H.264 MP4) and typically under ~4–6MB. Poster frames fall back to the existing images when video isn’t available.
